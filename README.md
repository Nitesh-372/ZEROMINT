# 🌍 Carbon Credit Platform — BlockChain based platform

A blockchain based platform for carbon credit project management, auditing, and marketplace trading.

---

## 📦 Stack Overview

### Frontend
- **React (Vite)** based SPA  
- **Axios** for REST API calls  
- Proxy `/api` → backend port **5055**  
- Environment config in `.env.development`

### Backend
- **Express + Node.js**  
- **MongoDB + Mongoose**  
- **JWT Authentication**  
- Optional **Blockchain integration** via `ethers.js`

---

## ⚙️ Setup

### 1. Prerequisites
- Node.js ≥ 18  
- MongoDB running locally or in Docker  

### 2. Environment Variables

**backend/.env**
